# 지능로봇공학 입문

# 개발환경

- ubuntu 22.04
- ros2 humble
- dynamixel wizard 2.0

# 하드웨어

- xm430 2개 (1축, 2축)
- ax18 2개 (3축, 4축)
- xc330 1개 (end effector)
- link와 base는 fusion360으로 설계 후 bambulab으로 출력

# 코드 실행
- ./dynamixel_control/draw_circle.py
- ./dynamixel_control/move_box.py

- "python3 filename" or "ros2 run packagename filename"
